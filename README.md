# סוגי קבצים והתוכן שלהם
בפרוייקט הזה שלנו יש 2 קבצי קלט שבהם אנחנו משתמשים

## קובץ רשימת הדיירים

קובץ ראשון נקרא "tennats-list.xlsx"
הקובץ מכיל רשימה של דיירים כשלכל דייר מופיע מספר דירה, קומה, שם בעלי הבית ומספר הטלפון שלו והמייל.
חלק מהדירות מושכרות. לדירות אלו יש גם מידע נוסף ברשומה. יופיעו שם שם שוכר הדירה, והטלפונים והאימייל שלו.

בסום הרשומה יש שלושה שדות שמתארים את התשלומים החודשיים שיש לבצע בדירה.
שדה אחד הוא וועד בית ושדה שני הוא קרן שיפוצים.
את קרן השיפוצים משלם בעל הבית בכל מקרה. את וועד הבית משלם בעל הבית רק אם הוא באמת גר בדירה. אם הדירה מושכרת אז השוכר ישלם את וועד הבית. בנוסף יש שדה שמתאר את שטח הדירה.

זהו הקובץ הראשון.
